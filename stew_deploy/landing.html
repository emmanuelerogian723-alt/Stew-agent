}

// Response:
{
  "api_key": "stew_xxxxxxxxxxxx",
  "plan": "free",
  "calls_limit": 1000

2. Ask STEW Anything (Real-Time Chat)

POST https://stew-agent.onrender.com/chat

{
  "message": "What is the latest AI news today?"
}

// Response — web-grounded with Serper search:
{
  "response": "Here are the latest AI developments...",
  "web_grounded": true,
  "success": true

2b. Execute Any Task (Full Agent Power)

POST https://stew-agent.onrender.com/task

{
  "task": "Research the top 5 AI startups in Africa and write a report"
}

// Response:
{
  "output": "Comprehensive report...",
  "success": true
}

3. Browse the Web

POST https://stew-agent.onrender.com/browse/navigate

{
  "url": "https://techcrunch.com",
  "question": "What are today's top stories?",
  "api_key": "stew_xxxxxxxxxxxx"
}

4. Connect an API (Universal Connector)

POST https://stew-agent.onrender.com/api/call

{
  "url": "https://api.openweathermap.org/data/2.5/weather",
