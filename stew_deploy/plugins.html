📖 What is MCP?
The Model Context Protocol (MCP) is an open standard for connecting AI agents to external tools and data sources. Once you connect an MCP server, STEW discovers its available tools and can use them in any task — no extra code needed.

Popular MCP servers: filesystem, GitHub, PostgreSQL, Slack, Notion, Google Drive, Stripe, and 200+ more.

STEW endpoint: POST https://stew-agent.onrender.com/mcp/connect
